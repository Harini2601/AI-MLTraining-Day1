# AI-MLTraining-Day1

# GenAI Day 1 Assignment

## What this does
- Accepts user input
- Generates text using OpenAI LLM
- Tests multiple temperature values (0.2, 0.7, 1.0)
- Saves results to JSON

## How to run
1. Install Python 3.9+
2. Add OPENAI_API_KEY in a .env file
3. Run:
   python generate_text.py
